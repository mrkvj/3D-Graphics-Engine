# 3d-graphics-engine
3d-graphics engine

About:
    3d-grahics engine implementation in C using OpenGL. 
    
    Uses GLFW for window, context, surfaces, inputs and events.  
    Uses GLEW for OpenGL extension loading.

Current Features:
    Open a window
    Draw a triangle


Progression log:
